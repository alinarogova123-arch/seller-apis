# API для продавцов
## seller.py
Скрипт запрашивает артикулы товаров OZON, запрашивает остатки часов Casio и на его основе появляется список с актуальными к продаже часам.

С помощью списка с акуальными к продаже часами, обновляет список выставленных на маркетплейс часов и обновляет для него цены.

## market.py
Скрипт запрашивает артикулы товаров Яндекс маркет, проверяет актуальность продаваемых товаров и их цены, затем обновляет товары и их цены на самом маркетплейсе,
сначала для модели работы FBS, затем аналогично для DBS.
